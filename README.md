## Installation

MailController is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
source 'https://github.com/Kulich-ua/PodSpecs.git'

platform :ios, '9.0'

target '<Your Target Name>' do

    pod 'MailController'

end
```
